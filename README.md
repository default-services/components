<br><br>

## 🐱‍🏍 Install

**npm:**
```bash
npm install @default-services/components
```
**yarn:**
```bash
yarn add @default-services/components
```
<br>

## 🐱‍💻 Use
```jsx
import React from 'react';
import { Button } from '@default-services/components';

const MyComponent = (props) => {
  return (
    <div>
      <Button>Hello Default!</Button>
    </div>
  );
};
```
